# MoneyControl Data Structure Analysis

## Key Findings

The MoneyControl page uses a NEW URL structure:
- Old URL: `https://www.moneycontrol.com/financials/tataconsultancyservices/results/quarterly-results/TCS`
- New URL (redirected): `https://www.moneycontrol.com/markets/financials/quarterly-results/tataconsultancyservices-TCS/`

## Table Structure

The quarterly data is in a table with this structure:

| Indicators | Trend | Sep '25 | Jun '25 | Mar '25 | Dec '24 | ... |
|------------|-------|---------|---------|---------|---------|-----|
